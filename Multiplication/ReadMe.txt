L'ALU, le decoblock et le séquenceur de cet architecture CPU sont générérs par le programme riscV.
Cet architecture CPU prend en compte l'ajout d'une instruction de multiplication REG-REG.
L'adresse de l'état ajouté est "11111".
L'opcode de cette nouvelle instruction est "1111111".
La valeur du signal funct 3 associé à ce nouvel état est "000".
La valeur du signal funct 7 associé à ce nouvel état est "0000000".
Le fichier Mem_In_Inst_64_Values.txt permet de tester lles instcution addi,multiplication et sw.
