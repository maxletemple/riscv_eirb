Ce programme esr fait pour générer le séquenceur et le décoblock.

Afin de générer le séquenceur,il est demandé à l’utilisateur de:
Définir l’état présent
Le nombre d’opcodes qui mènent à l’état suivant puis les états futurs
Les opcodes correspondant à ces états
Les conditions de passage à ces états
Les signaux de commande

Afin de générer le décoblock, il est demandé à l'utilisateur de : 
Insérer les codes des états en respectant l’ordre
